# Go Fundamentals 🚀

A comprehensive collection of **Golang (Go) fundamental concepts and examples** designed to help beginners understand the core building blocks of the Go programming language.

This repository is intended for **students, beginners, and developers** who want to learn Go from the basics with clear examples and simple explanations.

---

## 📚 About the Repository

This repository contains practical examples of **Go language fundamentals**, including:

* Basic syntax
* Variables and data types
* Control structures
* Functions
* Arrays and slices
* Maps
* Structs
* Packages
* Error handling
* File handling
* Small practice programs

Each example is written with **clean and readable code** to help learners understand Go concepts easily.

## ⚙️ Requirements

Make sure you have Go installed on your system.

Check installation:

```bash
go version
```

If Go is not installed, download it from the official website.

---

## ▶️ Running the Code

1. Clone the repository

```bash
git clone https://github.com/your-username/go-fundamentals.git
```

2. Navigate to the project folder

```bash
cd GOLANG
```

3. Run the program

```bash
go run main.go
```
